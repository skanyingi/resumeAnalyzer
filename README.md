# Resume Analyzer

An AI-powered career intelligence platform that analyzes, optimizes, and helps you land your dream role.

## Overview

Resume Analyzer is a modern web application that uses advanced AI to dissect, analyze, and optimize your professional resume. It goes beyond simple grammar checks to identify hidden strengths, strategic narrative gaps, and provides actionable insights to boost your chances of landing interviews at top-tier companies.

## Features

- **AI-Driven Strategic Insights**: Editorial-grade AI that understands career trajectories
- **Skill Extraction**: Automatically map your experiences to industry-standard taxonomies
- **Editorial Precision**: Line-by-line improvement suggestions focused on impact-oriented language
- **Benchmarking Analysis**: Compare your credentials against Fortune 500 candidates
- **Secure Document Handling**: Encrypted upload portal that respects your privacy

## Tech Stack

- Next.js 15
- React
- TypeScript
- CSS (Custom design system)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `src/app/` - Next.js app router pages
- `src/components/` - Reusable UI components
